Cursor 怎么设置中文？
界面汉化与 AI 对话设置详解

Cursor 默认是全英文界面，但这并不影响我们使用。通过简单的设置，你可以轻松将界面汉化，并让 AI 始终用中文回答你的问题。

2026-01-30
Cursor-VIP Team

汉化指南概览
设置 Cursor 中文主要分为两个部分：一是 界面语言 （菜单、按钮等），二是 AI 回复语言 （Chat、Cmd+K 等）。你可以根据自己的需求选择只设置其中一项，或者全部设置为中文。

一、设置中文界面 (界面汉化)

由于 Cursor 是基于 VS Code 开发的，所以我们可以直接利用 VS Code 的中文语言包插件来实现汉化。

1

安装中文语言包

打开 Cursor，点击左侧活动栏的 扩展图标 （Extensions），或者直接按下快捷键 Cmd/Ctrl + Shift + X 。

在搜索框中输入 Chinese 或 简体中文 。

找到由 Microsoft 发布的 "Chinese (Simplified) Language Pack for Visual Studio Code" 插件，点击 Install 按钮。

2

切换显示语言

安装完成后，按下快捷键 Cmd/Ctrl + Shift + P 打开命令面板。

在搜索框输入 Configure Display Language ，然后回车。

在弹出的列表中选择 zh-cn 或 中文(简体) 。

3

重启生效

Cursor 会提示需要重启 (Restart) 才能应用更改。点击重启按钮后，你的 Cursor 界面就会变成中文啦！

二、设置中文聊天 (AI 回复)

即使界面是英文，你也可以强制要求 AI 始终用中文回答你的问题。这对于习惯英文界面但需要中文解释代码的开发者非常有用。

  • 点击右上角齿轮图标，进入 Cursor Settings 。
  • 找到 General -> Rules for AI 模块。
  • 在 "User Rules" 输入框中添加以下规则：
始终使用中文回复我的问题。
Always reply in Chinese.

添加后，AI 无论在 Chat 还是 Cmd+K 中，都会优先使用中文回答。

温馨提示

官方中文文档

Cursor 官方文档已经支持中文版。如果你在使用过程中遇到复杂问题，可以直接去查阅官方文档，内容非常详细。

VS Code 键位映射

如果你是 VS Code 的老用户，可以在设置中搜索 "Keymap"，将键盘映射方案切换为 "VS Code"，这样所有快捷键就和你熟悉的一模一样了，无缝切换。
